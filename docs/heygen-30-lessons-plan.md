# HeyGen — Plan de production des 30 leçons Nexum AI School

## Workflow de production
```
1. Script rédigé dans Notion
2. HeyGen : coller script → choisir avatar Leandro → voix FR → générer
3. Télécharger MP4
4. Héberger (Google Drive ou Vimeo gratuit)
5. Coller le lien dans le salon Discord #module-XX
```

---

## Ordre de production recommandé

Commencer par M01 (module d'entrée, donne le ton) → M02 → M03 dans l'ordre.
Objectif : 2 leçons produites par jour = formation complète en 15 jours.

## Script HeyGen — Template pour chaque leçon

Copier-coller ce template dans HeyGen, adapter le contenu :

```
[INTRO — 15 secondes]
"Salut, c'est Leandro. Dans cette leçon, on va [OBJECTIF DE LA LEÇON EN UNE PHRASE]."

[ACCROCHE — 20 secondes]
"Voilà pourquoi c'est important pour toi : [BÉNÉFICE CONCRET]. Et à la fin de cette vidéo, tu sauras exactement comment faire."

[CONTENU — 70% de la durée]
[Point 1] "D'abord, [CONCEPT]. Voici comment ça marche..."
[Point 2] "Ensuite, [CONCEPT]. L'erreur que font la plupart des gens ici, c'est..."
[Point 3] "Enfin, [CONCEPT]. La technique que j'utilise personnellement..."

[DÉMONSTRATION — si applicable]
"Regarde, je te montre directement..."

[EXERCICE — 30 secondes]
"Ton exercice pour cette leçon : [EXERCICE SIMPLE ET CONCRET]. Tu as 15 minutes pour le faire."

[OUTRO — 15 secondes]
"Voilà pour cette leçon. Dans la prochaine, on verra [SUJET LEÇON SUIVANTE]. N'hésite pas à poser tes questions dans le salon [#MODULE-XX] du Discord. À tout de suite."
```

## Paramètres HeyGen recommandés

| Paramètre | Valeur |
|-----------|--------|
| Avatar | Choisir un avatar masculin européen (ou créer avatar custom Leandro) |
| Voix | Français — Matthieu ou Hugo (voix naturelle) |
| Vitesse | 0.95 (légèrement plus lent que par défaut) |
| Format | 16:9 pour YouTube/Discord, 9:16 pour TikTok/Reels |
| Résolution | 1080p |
| Background | Fond sombre uni (#040407) ou bureau stylisé |
